## Project Overview

This project develops machine learning models to predict the ten-year risk of coronary heart disease using the Framingham Heart Study dataset.

The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, handling missing values, class balancing using SMOTE, model development, evaluation, and interpretation.

Research Questions

1. Which demographic, behavioural, and clinical factors are most strongly associated with Ten-Year Coronary Heart Disease (CHD)?

2. Which machine learning model provides the best predictive performance for identifying patients at risk of CHD?

3. Which variables contribute most to the prediction of heart disease?
